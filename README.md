# garethstretton.com

Personal site for Gareth Stretton — custom software for small &amp; mid-sized businesses.

A static, single-page site. No build step, no dependencies.

- `index.html` — the whole site (inline CSS, no external requests)
- `face.png` — headshot used in the "Why work with me" section

## Local preview

Open `index.html` in a browser, or serve the folder:

```
python3 -m http.server 8000
```

then visit http://localhost:8000
